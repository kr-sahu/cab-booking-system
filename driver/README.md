# 🚗 Zuber Driver Portal

This module allows drivers to manage ride requests, view trip details, and update their status.

---

## 🚀 Getting Started

1. Apply for driving:
   http://localhost/cab_app/driver/apply.php  

2. Wait for admin approval  

3. Login:
   http://localhost/cab_app/driver/login.php  

---

## ✨ Features

- Receive and accept ride requests  
- View pickup and drop routes on map  
- Track trip history and earnings  
- Update profile details  
- Toggle Online / Offline availability  

---

## 🔐 Note

Drivers can only access the dashboard after admin approval.
