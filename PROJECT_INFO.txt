------------------------------------------------------------------------
[ DATABASE CONFIGURATION ]
------------------------------------------------------------------------
Database Name: cab_booking
SQL File: database.sql (located in root directory)
Connection Config: api/db_connect.php
Recommended PHP Version: 7.4 or above

------------------------------------------------------------------------
[ SYSTEM ACCESS & CREDENTIALS ]
------------------------------------------------------------------------

1. ADMIN PANEL
   URL:      http://localhost/cab_app/admin/login.php
   Username: admin
   Password: Password@123
   * Used to approve drivers, manage bookings, and monitor system.

2. DRIVER PORTAL
   URL:      http://localhost/cab_app/driver/login.php
   * Workflow:
     a) Driver applies via /driver/apply.php
     b) Admin approves the application
     c) Driver logs in using registered credentials

3. CUSTOMER PORTAL
   URL:      http://localhost/cab_app/
   * Users can register/login and book rides.

------------------------------------------------------------------------
[ SETUP & INSTALLATION ]
------------------------------------------------------------------------
1. Import 'database.sql' into MySQL.
2. Configure database credentials in 'api/db_connect.php'.
3. Ensure 'uploads/' folder has read/write permission.
4. Internet required for CDN libraries (Tailwind, Leaflet).
