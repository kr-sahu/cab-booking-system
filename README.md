# 🚖 Zuber - Cab Booking & Management System

A web-based cab booking system built using PHP and MySQL.  
It includes separate panels for users, drivers, and admin to manage the complete ride process.

---

## 🏗️ System Overview

### 👤 User Panel
- Book rides easily  
- View booking history  
- Manage profile  

---

### 🛠️ Admin Panel (`/admin`)
- Approve driver applications  
- Manage bookings and users  
- View system activity  

---

### 🚗 Driver Panel (`/driver`)
- Accept and complete rides  
- View trip history  
- Check profile and status  

---

## 🚀 Features

- 📍 Live route display using Leaflet.js  
- 📱 Fixed country code (+91) for mobile input  
- 🔔 Booking notifications system  
- 🧾 Ride history tracking  
- 🔐 Login & authentication system  

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript, Tailwind CSS  
- Backend: PHP  
- Database: MySQL  
- Maps: Leaflet.js  

---

## ⚙️ Setup

1. Install XAMPP  
2. Import `database.sql` into MySQL  
3. Place project in `htdocs`  
4. Start Apache & MySQL  
5. Open:  
   http://localhost/cab_app  

---

## 📁 Project Structure

- `/admin` – Admin panel  
- `/api` – Backend APIs  
- `/driver` – Driver panel  
- `/layout` – Common UI components
