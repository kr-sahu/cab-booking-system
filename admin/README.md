# 🛠️ Admin Panel

This module is used to control and manage the entire cab booking system.

---

## 🚀 Features

### 📊 Dashboard
- View total bookings, users, drivers, and cabs  

### 🚗 Booking Management
- View all ride requests  
- Assign drivers and cabs  
- Update ride status  

### 🧑‍✈️ Driver Management
- Approve or reject driver applications  
- Manage driver availability and status  

### 🚘 Cab Management
- Add, update, or remove cabs  
- Set cab status (Available, Busy, Maintenance)  

### 👤 User Management
- View registered users  
- Block/unblock users if required
